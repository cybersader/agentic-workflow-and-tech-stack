Skills are loaded INTO an agent's context at invocation time. Unlike agents (which spawn as separate contexts), skills are *passive expertise* — Claude reads each skill's `description` field and decides autonomously when to apply it.

See [skills-vs-agents principle](../principles/03-skills-vs-agents/) for the distinction.

## Listed here

The sidebar autogenerates from this directory. Each page is one skill extracted from its subdirectory's `SKILL.md`.
