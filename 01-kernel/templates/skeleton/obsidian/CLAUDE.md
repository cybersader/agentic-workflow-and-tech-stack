## Purpose

Personal knowledge management using Obsidian's linking and tagging features.

## Structure

| Folder | Purpose |
|--------|---------|
| `Inbox/` | Quick captures, unprocessed notes |
| `Notes/` | Processed, linked thoughts |
| `Projects/` | Active project documentation |
| `Templates/` | Templater/core templates |

## Workflow

```
1. Capture → Inbox/
2. Process → Move to Notes/ or Projects/
3. Link → Add wikilinks to related notes
4. Tag → Apply appropriate tags
```

## Key Features

- **Wikilinks:** `[[Note Name]]` for connections
- **Tags:** Nested tags like `#topic/subtopic`
- **Frontmatter:** YAML metadata at top of notes
- **Templates:** Use for consistent note structure

## SEACOW Mapping

- **Capture:** Inbox/, Daily Notes
- **Work:** Notes/, Projects/
- **Output:** Published/ (if using Obsidian Publish)
- **System:** Templates/, .obsidian/
- **Relation:** Wikilinks, tags, graph view
